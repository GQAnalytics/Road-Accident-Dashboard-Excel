# Road Accident Dashboard (Excel Project)

## Project Overview
This dashboard analyzes road accident data using Microsoft Excel.

## Features
- Total Casualties Analysis
- Severity Analysis
- Vehicle Type Analysis
- Monthly Trend Comparison
- Road Type Analysis
- Road Surface Analysis
- Urban vs Rural Casualties
- Weather Condition Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
